# URL Shortener

A URL Shortener application developed using Laravel 12 as part of a Backend Developer Assignment.



## Features

1. Authentication (Login & Logout)
2. Dashboard
3. Company Management (CRUD) - Accessible only by Super Admin
4. Invitation Management
5. URL Shortener
6. Public URL Redirection



## Technologies Used

- PHP 8.2
- Laravel 12
- HTML + Bootstrap 5.3
- MySQL



## Installation

1. Clone the repository.
2. Configure the ".env" file.
3. Run 'php artisan migrate --seed'.
4. Start the application using 'php artisan serve'.
